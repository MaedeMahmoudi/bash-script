## 🚀 Quick Start

### Prerequisites
- Linux/macOS/WSL (Windows Subsystem for Linux)
- Bash 4.0 or higher
- Basic terminal familiarity

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MaedeMahmoudi/bash-script.git

    Navigate to the project:
    bash

    cd bash-script

    Explore the content:

        Read the book: book/bash-scripting-guide.pdf

        Run examples: cd code/chapter-01/ && bash hello-world.sh

        Practice exercises in each chapter

    Start coding:
    bash

    # Create your first script
    nano my-first-script.sh
    chmod +x my-first-script.sh
    ./my-first-script.sh

🎓 Learning Path
Chapter	Topic	What You'll Learn
01	Introduction	Bash basics, shell types, first script
02	Variables	Data types, arrays, parameter expansion
03	Conditionals	If-else, case statements, test commands
04	Loops	For, while, until loops, break/continue
05	Functions	Creating reusable code, scope, recursion
06	Advanced	Sed, awk, process management, signals
07	Real Projects	Backup scripts, monitoring tools, automation
🛠️ Usage Examples
Run a script:
bash

bash script-name.sh
# or
./script-name.sh  # (make executable with chmod +x first)

Test your code:
bash

bash -n script.sh    # Syntax check
bash -x script.sh    # Debug mode with step-by-step execution

Get help:
bash

man bash
help [command]
