# کتاب آموزش اسکریپت نویسی Bash

یک منبع کامل و کاربردی برای یادگیری اسکریپت‌نویسی در پوسته‌ی Bash، همراه با مثال‌های عملی و کدهای آماده.

---

## ساختار پروژه

bash-script/
├── book/
│ ├── bash-scripting-guide.pdf
│ └── bash-scripting-guide.odt
└── code/
├── chapter-01/
├── chapter-02/
└── ...
text


- **book/**: شامل فایل‌های کتاب به دو فرمت PDF و ODT برای مطالعه‌ی آسان.
- **code/**: شامل کدهای عملی هر فصل که می‌توانید آن‌ها را اجرا و تست کنید.

---

## نحوه استفاده

برای شروع کار، ریپازیتوری را روی سیستم خود کلون کنید:

```bash
git clone https://github.com/MaedeMahmoudi/bash-script.git
cd bash-script

سپس می‌توانید:

    فایل‌های کتاب را در پوشه‌ی book/ باز کنید و مطالعه نمایید.

    کدهای هر فصل را در پوشه‌ی code/ مشاهده و با دستور bash filename.sh اجرا کنید.
